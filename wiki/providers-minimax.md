# MiniMax
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key in the [MiniMax platform](https://platform.minimax.io/). The default is `MiniMax-M2.7`; use model discovery for the current catalog. GitMind sends the selected diff and prompt to MiniMax. Confirm regional endpoint access, credentials, and quota after errors.
