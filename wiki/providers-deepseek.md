# DeepSeek
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key on the [DeepSeek platform](https://platform.deepseek.com/api_keys). The default is `deepseek-v4-flash`; discovery shows available chat and reasoning models. GitMind sends the selected diff and prompt to DeepSeek. Check credentials, model access, and quota on failures.
