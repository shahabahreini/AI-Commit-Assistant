# Hugging Face
> Verified against GitMind `5.0.2` on June 7, 2026.

Use a [Hugging Face access token](https://huggingface.co/settings/tokens). The default model is `mistralai/Mistral-7B-Instruct-v0.3`; hosted model availability varies. GitMind sends the selected diff and prompt to the configured inference service. Confirm the token can access the selected model when setup checks fail.
