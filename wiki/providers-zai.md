# Z.ai
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key at [Z.ai](https://z.ai/). The default model is `glm-5.1`; `gitmind.zai.endpoint` selects the `regular` or default `coding` endpoint. **Load Z.ai Models** refreshes discovery. GitMind sends the selected diff and prompt to Z.ai; verify the endpoint matches the key and account.
