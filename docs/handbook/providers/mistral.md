# Mistral AI
> Verified against GitMind `5.0.2` on June 7, 2026.

Create an API key in the [Mistral console](https://console.mistral.ai/). The default is `mistral-small-4`; **Load Mistral Models** refreshes the searchable catalog. GitMind sends the selected diff and prompt to Mistral. Check credentials, model access, and provider quota when requests fail.
