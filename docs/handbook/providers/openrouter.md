# OpenRouter
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key at [OpenRouter](https://openrouter.ai/keys). The default is `google/gemma-3-27b-it:free`; searchable discovery exposes OpenRouter's changing multi-provider catalog. GitMind sends the selected diff and prompt through OpenRouter and the routed model provider. Review OpenRouter routing and privacy policies.
