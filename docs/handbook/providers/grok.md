# xAI Grok
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key in the [xAI console](https://console.x.ai/). The default is `grok-4.3`; model discovery reflects account access. GitMind sends the selected diff and prompt to xAI. Check API access, selected model, and provider limits when setup fails.
