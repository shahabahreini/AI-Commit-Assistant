# Google Gemini
> Verified against GitMind `5.0.2` on June 7, 2026.

Use an API key from [Google AI Studio](https://aistudio.google.com/app/apikey). The default model is `gemini-3.1-flash`; use model discovery for the current catalog. GitMind sends the selected diff and prompt to Google. Check API Setup first; account quota failures are not retried automatically.
