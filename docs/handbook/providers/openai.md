# OpenAI
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key on the [OpenAI API platform](https://platform.openai.com/api-keys). The default is `gpt-5.5-instant`; load models to see those available to the account. GitMind sends the selected diff and prompt to OpenAI. Billing, project permissions, and rate limits are controlled by OpenAI.
