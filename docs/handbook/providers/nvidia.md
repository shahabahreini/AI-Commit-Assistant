# NVIDIA Hosted NIM
> Verified against GitMind `5.0.2` on June 7, 2026.

Create a key at [NVIDIA Build](https://build.nvidia.com/models). The default is `meta/llama-3.3-70b-instruct`; **Load NVIDIA Models** refreshes hosted NIM discovery. GitMind sends the selected diff and prompt to NVIDIA's hosted API. Confirm the model is hosted and available to the key.
