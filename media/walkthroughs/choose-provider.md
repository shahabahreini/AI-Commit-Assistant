# Choose your AI provider

GitMind generates commit messages with the provider **you** choose. Bring your own key, run a local Ollama model, or reuse your GitHub Copilot subscription.

GitMind includes **17 built-in providers**:

- **OpenAI**, **Anthropic**, **Google Gemini**, **NVIDIA NIM**, **MiniMax**
- **DeepSeek**, **xAI Grok**, **Groq**, **Perplexity**, **Z.ai**
- **Mistral**, **Cohere**, **Together AI**, **OpenRouter**, **Hugging Face**
- **Ollama** for local generation and **GitHub Copilot** for existing subscribers

GitMind Pro also adds a configurable **Custom API** provider.

Open **Settings > Model Settings**, select a provider, then search or load its available models.
