# Add your API key

Most cloud providers need an API key. Grab one from your provider, then paste it into GitMind's **Settings → Model Settings**.

Common places to get a key:

- **OpenAI** — https://platform.openai.com/api-keys
- **Anthropic** — https://console.anthropic.com/
- **Google Gemini** — https://aistudio.google.com/app/apikey
- **NVIDIA NIM** — https://build.nvidia.com/models
- **DeepSeek** — https://platform.deepseek.com/api_keys
- **Mistral** — https://console.mistral.ai/api-keys/
- **OpenRouter** — https://openrouter.ai/keys

After entering a key, use the model loader and searchable picker to select an available model.

Prefer no key at all? Choose **Ollama** to run a model locally, or **GitHub Copilot** to reuse your existing subscription.

> Tip: GitMind Pro can **encrypt** your stored API keys for extra safety.
