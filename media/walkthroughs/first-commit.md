# Generate your first commit message

1. Stage some changes in the **Source Control** panel.
2. Click the **GitMind** icon in the Source Control toolbar (or run **Generate GitMind Commit Message** from the Command Palette).
3. GitMind analyzes your diff and writes a clear, conventional commit message right into the message box.

You can pick a **commit style** (Conventional, Semantic, Descriptive, Gitmoji, and more) in Settings, and add custom context to any generation.

That's it — commit as usual. 🎉
